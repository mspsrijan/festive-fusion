- C: `undefined`

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: ?

<i>**C: "12"**
We are adding a number with a string. In this case string will get the priority, so the sum function will concate string 1 with string 2, thus the result will be string 12. However, it will not print anything on console as we do not have a console log here.</i>

</p>
</details>
